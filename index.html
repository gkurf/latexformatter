About

This formatter tool converts plaintext equations and algorithms into LaTeX format, specifically designed for Gradescope assignments with LaTeX enabled and similar platforms.

Basic Usage

  • Type or paste your content in the left panel
  • Use the toolbar buttons below to insert common LaTeX symbols and formatting
  • The right panel shows a live preview of how your content will appear
  • Click on the preview area to see the raw LaTeX code
  • Use the "Copy Raw Output to Clipboard" button to copy the formatted LaTeX

Algorithm Auto-Format Features

When "Algorithm Auto-Format" is enabled, the formatter automatically:

  • Bolds keywords ()
  • Formats text elements ()
  • Auto-indents after loop keywords

Example Algorithm

Below is an example of a BubbleSort algorithm shown as typed in plain text vs. its formatted output:

Plain Text Input
algorithm BubbleSort(A:array)
	let n be the size of A
	repeat ← True
	while repeat do
		repeat ← False
		for i from 0 to n-2 do
			if A[i] > A[i+1] then
				Swap(A, i, i+1)
				repeat ← True
			end if
		end for
	end while
	return A
end algorithm
